# VOLT — Electronics & Tech Shopify Theme

A high-performance custom Shopify theme built for electronics and tech stores.

## Features

- Built on Dawn 14.x base
- Metafields-powered product spec tables
- Judge.me reviews integration
- localStorage wishlist with persist
- Section Rendering API cart (no page reload)
- Product comparison (up to 3 items)
- Recently viewed products
- Subscription product UI
- Sticky Add to Cart bar
- Full WCAG accessibility compliance
- Light / dark mode support
- SEO optimized with JSON-LD structured data

## Tech Stack

- Shopify Liquid
- Vanilla JS (no jQuery)
- CSS custom properties
- Shopify Section Rendering API

## Setup

1. Clone the repo
2. Install Shopify CLI
3. Run `shopify theme dev --store your-store.myshopify.com`

## Development workflow

- `main` — production ready
- `develop` — integration branch
- `feature/*` — individual features
